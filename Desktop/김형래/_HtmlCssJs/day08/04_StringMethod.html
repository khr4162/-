가장 많이 다루는 데이터 타입 : string

  • 문자열 데이터는 index(0부터 시작)를 가지고 있음.
  • 문자열 데이터 값 내에서는 개행(br)하는 것을 추천하지 않음
  • new string(); 객체 타입으로 선언은 가능하나 추천하지 않음
  • length : 문자열의 총 길이 => index는 length-1

문자관련 메서드 : java에서 사용하면 string 메서드와 비슷 (다른 프로그래밍 언어도 비슷)

  • 문자 추출 관련 메서드 : substing(), substr(), slice()
  • 문자 대체 메서드 : replace(oldstr, newstr) : 첫번째 매칭되는 문자만, 대소문자 구분
  • trim() : 문자열 양측 공백 제거
  • padstart(목표문자길이, 채울문자) / padend : 앞쪽에서 / 뒷쪽에서 채우기
    문자 값의 힌트, 010101-*******
  • charat(index) : 해당 index 문자 추출
  • indexof("str") : 문자열의 위치를 반환
  • startwith("str") / endwith("str") :
    지정 문자열이 탐색대상 문자열에서 시작/끝에 존재하는지 확인하는 메서드
