이벤트 객체는 전역객체로써 모든 태그에 객체를 참조시킬수 있다.

  • a 태그와 button 태그에 이벤트를 주로 참조시켜 클릭에 대한 반응을 기대할 수 있게 만드는 구조가 일반적임
  • a태그는 href 속성에 의해 자체 이동이 구현되어 있어서 이동을 막는 코드가 필요
  • 이벤트 객체를 부여하는 방식 3가지. 각자 목적과 작동방식이 다르다.
  • 화면 출력 영역 document영역(객체)라고 한다
  • 태그는 html element, 속성 attribute, 일반 텍스트는 textNode라고 한다
  • html 태그로 출력 : innerHTML라는 property를 사용
  • text로 출력 : innerText라는 property를 사용
  • 객체의 값을 가져올 때 (id) : getElementById('id명')
  • 객체의 값을 가져올때 (class) : querySelector('selector')

event - 데이터 받기

form 태그를 통해 데이터 받아오기

  • input, textarea, select, heml element의 속성, 텍스트 데이터를 받아오기
  • form 태그의 데이터는 value 속성으로 값이 매핑되어 있으므로 value 속성을 호출하면 된다.
  • select option, radio, checkbox value는 선택된값을 가져와야 하기 때문에 속성에 value 값을 추가하는 작업이 필요
  • value속성의 데이터는 모두 string 처리된다.
